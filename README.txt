Para a execução do projeto:
	1 - Carregue o projeto em uma IDE com suporte ao JAVA (netbeans ou eclipse);
	2 - Build o projeto, e logo abaixo pelo console repare que o programa solicita o arquivo de entrada;
	3 - Informe o caminho de entrada e pressione ENTER;
	4 - Informe qual Heuristica adotar ManHattan[M-m]||Euclidiana[E-e]||Camberra[C-c];
	5 - E logo após será printado o resultado no console junto com a analise;

OBS:
	Os aquivos de Entrada são as configuraões iniciais do tabuleiro Puzzle_8, no .zip já contem algumas 
	configurações inicias porem pode-se adicionar outras configurações desde que não saia do formato de uma 
	matriz 9x9 com valores de '0' à '8' tendo como '0' o espaço vazio.
	Tambem é predefinido o formato do objetivo final do Puzzle_8 como sendo o arquivo "objetivo.txt" que
	tambem pode ser alterado desde que obedeça as regras anteriores ditas.
